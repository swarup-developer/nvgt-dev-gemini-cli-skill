## Description
These are resources, libraries, and plugins that you can integrate into your game.

Note, not all libraries are placed in this folder; instead, I will provide GitHub links for you to easily clone them, and any that are not on GitHub will be placed in this folder.

## Resources

* king_audio_engine, this is a library/plugin that helps you write an audio system for your game; it's very good for 3D/HRTF audio. The king_audio_engine folder is located in this directory.
